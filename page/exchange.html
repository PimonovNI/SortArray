Сортування обміном

Також роздивимось ще один простий і поширений метод – це сортування обміном, а саме сортування "бульбашкою" і "гребінцем".

Що з себе представляє цей метод? Порівнюють два елементи шо знаходяться поряд і, для сортування по зростанню наприклад, якщо більший елемент лівіше, то його змінюють з правим елементом.




Сортування "бульбашкою"

Тепер детальніше. Беремо масив . array[5] , з елементами .3 64 12 4 1.. І наше завдання розташувати його елементи по зростанню. Використаємо сортування "бульбашкою".

Створимо дві змінні int temp,k; де temp – змінна для обміну значень масиву, а k – параметр, який буде зменшуватись з кожною ітерацією і буде вказувати на кількість повторення циклу (початкове значення дорівнює довжині масиву).

Створюємо цикл:

for (int i = 1; i< array.length;i++){}

Він вказує на кількість усіх повторень, при чому він починається з одиниці, бо немає сенсу порівнювати один і той же елемент.


Далі діло за малим: створюємо з параметром, який спочатку дорівнює нулю і менше k. У тілі циклу перевіряємо: "Чи не більше попередній елемент, ніж теперішній?". Якщо так міняймо їх місцями. Після виконнаня циклу зменшуймо k на одиницю:

for (int j = 1;j < k;j++){

if (array[j] < array[j-1]) {

temp=array[j-1];

array[j-1]=array[j];

array[j]=temp;

}

}

k--;

Тепер розглянемо реалізацію цього алгоритму на таких мовах програмування, як Java і C++:


Java


		import java.util.Arrays;

public class Main {

    public static void main(String[] args) {
	int []array = {3, 64, 12, 4, 1};
        System.out.println(Arrays.toString(array));
	int temp, k= array.length;
	for (int i = 1; i< array.length;i++){
	    for (int j = 1;j < k;j++){
	        if (array[j] < array[j-1]) {
	            temp=array[j-1];
	            array[j-1]=array[j];
	            array[j]=temp;
	        }
        }
	    k--;
    }
        System.out.println(Arrays.toString(array));
    }
}

Сортування "гребінцем"

Сортування"гребінцем" більш ефективне, ніж сортування "булбашкою". В виконанні він трохи складніше. Він представляє собою промодифіковану версію сортування "бульбашкою", тобто спочатку звіряють з меншого елементу до більшого, а потім з більшого до меншого.


Для виконання цього алгоритму створимо логічну змінну b, в ній буде зберігатися значення, що означає: "Чи проходили обміни?" - для вихода з циклу.

Тепер розглянемо реалізацію цього алгоритму на таких мовах програмування, як Java і C++:


Java


		import java.util.Arrays;

public class Main {

    public static void main(String[] args) {
        int array[] = {3,64, 12, 4, 1};
        System.out.println(Arrays.toString(array));
        boolean b;
        int temp;
        for (int i = 0; i < array.length / 2; i++) {
            b=false;

            for (int j = i; j < array.length - i - 1; j++)
            {
                if (array[j] > array[j + 1])
                {
                    temp = array[j];
                    array[j] = array[j + 1];
                    array[j + 1] = temp;
                    b = true;
                }
            }

            for (int j = array.length - 2 - i; j > i; j--)
            {
                if (array[j - 1] > array[j])
                {
                    temp = array[j];
                    array[j] = array[j - 1];
                    array[j - 1] = temp;
                    b = true;
                }
            }
            if (!b) break;
        }
        System.out.println(Arrays.toString(array));
    }
}



C++

		
void shekerSort(double *mass, int count)
{
  int left = 0, right = count - 1;
  int flag = 1; 
  while ((left < right) && flag > 0)
  {
    flag = 0;
    for (int i = left; i < right; i++) 
    {
      if (mass[i]>mass[i + 1]) 
      {          
        double t = mass[i];
        mass[i] = mass[i + 1];
        mass[i + 1] = t;
        flag = 1;  
      }
    }
    right--; 
    for (int i = right; i>left; i--) 
    {
      if (mass[i - 1]>mass[i])
      {      
        double t = mass[i];
        mass[i] = mass[i - 1];
        mass[i - 1] = t;
        flag = 1;
      }
    }
    left++;
  }
}
